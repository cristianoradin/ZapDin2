✏️ Configurar Mensagem Padrão

Monte o layout da mensagem que será enviada ao cliente após cada venda. Use as variáveis abaixo para incluir dados dinâmicos do ERP.

Variáveis disponíveis:
{nome} Nome do cliente  ·  {telefone} Telefone  ·  {data} Data da compra
{produtos} Lista de produtos (nome, qtd, valor unitário)
{valor_total_itens} Valor total dos itens  ·  {valor_total} Valor total da compra

📱 Números WhatsApp

Cada sessão representa um número conectado para envio de mensagens.

Como conectar: Clique em Mostrar QR na sessão → Abra o WhatsApp no celular → toque em ⋮ Menu → Aparelhos conectados → Conectar aparelho → aponte a câmera para o QR Code.

📚 Documentação da API

Todos os endpoints abaixo exigem o header x-token: SEU_TOKEN.
O token é gerado em Integrações → Token API.

📎 Documento completo para o integrador ERP: Abrir / Imprimir como PDF →
⚠️ Campo telefone: envie apenas DDD + número (sem o DDI 55). O sistema adiciona o 55 automaticamente.
Exemplo: envie "11999990000" — o sistema usa "5511999990000".

🔐 Autenticação

Todas as requisições ao ERP devem incluir o token no header HTTP:
x-token: rYq1OWgra44Nx_nlaSWF...

O token é gerado automaticamente em Token API → ⚡ Gerar Novo Token.
Respostas de erro possíveis: 401 Token inválido · 503 Sem sessão WhatsApp ativa

POST /api/erp/venda

Recebe os dados da venda e envia mensagem WhatsApp ao cliente usando o template configurado.

CampoTipoObrigatórioDescrição
telefonestringSimDDD + número — sem DDI, o sistema insere o 55. Ex: "11999990000"
nomestringSimNome completo do cliente
produtosarrayNãoLista de itens — ver estrutura abaixo
valor_total_itensstringNãoSoma dos itens (ex: "179,70")
valor_totalstringNãoTotal final da compra (com descontos/taxas)
datastringNãoData da compra — padrão: data atual
mensagem_customstringNãoSubstitui o template configurado no painel

Estrutura de cada item em produtos:

CampoTipoDescrição
nomestringNome do produto/serviço
quantidadenumberQuantidade consumida
valor_unitariostringValor unitário (ex: "5,99")

Exemplo completo:

POST /api/erp/venda
x-token: SEU_TOKEN
Content-Type: application/json

{
  "telefone": "11999990000",
  "nome": "João Silva",
  "produtos": [
    { "nome": "Gasolina Comum", "quantidade": 40, "valor_unitario": "5,99" },
    { "nome": "Óleo Motor 5W30", "quantidade": 1, "valor_unitario": "42,00" }
  ],
  "valor_total_itens": "281,60",
  "valor_total": "281,60",
  "data": "21/04/2026"
}

Resposta de sucesso: 200 {"ok": true, "sessao": "a1b2c3d4"}

POST /api/erp/arquivo

Envia um arquivo ao cliente via WhatsApp. O conteúdo deve ser enviado em base64.

CampoTipoObrigatórioDescrição
telefonestringSimDDD + número — sem DDI, o sistema insere o 55. Ex: "11999990000"
nome_arquivostringSimNome com extensão (ex: "nota_fiscal.pdf")
conteudo_base64stringSimConteúdo do arquivo codificado em base64
mensagemstringNãoLegenda do arquivo
 
POST /api/erp/arquivo
x-token: SEU_TOKEN
Content-Type: application/json

{
  "telefone": "11999990000",
  "nome_arquivo": "nota_fiscal_001.pdf",
  "conteudo_base64": "JVBERi0xLjQKJ...",
  "mensagem": "Segue sua nota fiscal eletrônica."
}

3. Copie o Token gerado e cole no campo acima
4. Adicione o bot ao grupo/canal desejado e envie uma mensagem
5. Use @userinfobot ou a API do Telegram para obter o Chat ID
6. Cole o Chat ID, salve e clique em Testar Conexão
