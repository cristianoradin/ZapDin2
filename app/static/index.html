📚 Documentação da API

Endpoints disponíveis para integração com o ERP.

MétodoEndpointDescrição
POST/api/erp/vendaEnvia mensagem de venda via WhatsApp
POST/api/erp/arquivoEnvia arquivo PDF em base64
GET/api/arquivosLista arquivos enviados
GET/api/statsEstatísticas do painel

Exemplo — POST /api/erp/venda

{
  "telefone": "5511999990000",
  "nome": "João Silva",
  "valor": "R$ 150,00",
  "data": "20/04/2026"
}

Header obrigatório: x-token: SEU_TOKEN
